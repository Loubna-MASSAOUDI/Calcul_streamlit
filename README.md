# Calcul_streamlit
The name of this project is a data calculator and visualization using Streamlit 
This app has an objective of calculating the financial ratio of a specific dataset with its visualisation. 
This projects used python and streamlit technologies.
This project can be of use only with the right dataset. You can see a copy of the dataset I used in data.xlsx
